# BigdataApp1
这个项目是一个改写`Oracle存储过程`为`SparkApp`的一个应用

##主要的功能
* syrk里面是`统计`，`数据质量检查`等
* jf里面是`人员积分`
* ps:jf里面的代码可读性更好，最近实现的，而syrk比较久远了

##主要思路
* 把每个`实体`的各项指标`join`在一起，然后map
* 这个做法与sql中的`游标中不断select`的在做法不予苟同
（详见jf里面的存储过程的sql文件）
* 但是`性能`上有很大提升

##主要技术
* 主要使用了`SparkSql` 和`SparkCore`库
